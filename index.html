Funções em JS

Funções são um conjunto de instruções que realizam uma tarefa.

Função com return

O return finaliza uma função e retorna um valor para onde essa função foi chamada.

A diferença do return para o console.log é que o console.log apenas exibe os valores na tela, inclusive o valor retornado para o return.

Switch

É uma estrutura condicional. Avalia uma expressão, combinando o valor da expressão para a cláusula case, e executa as instruções associadas ao case. Se nenhum instrução for verdadeira a condição será exibida no default que é uma palavra reservada; O comando break é importante para que não exiba todas as mensagens no console, ele vai literalmente PARAR na condição que você colocar.
